# LeetCode

每日算法练习

每个文件夹是一个题目，文件夹命名格式为 `2020-03-23_leetcode-problemNumber-problemName` ，

在readme中：

1. 需要简洁但清楚描述题目（能描述清楚题目本质上是加深理解，所以很有必要）；
2. 说清核心的算法和逻辑；
3. 时间、空间复杂度。

每做一道题需要回顾思考以上三点，而不简简单单是AC就完事拍拍屁股不管了，那样收获很小，相当于没做，花那么多时间在这上面岂不是白白浪费了吗。
